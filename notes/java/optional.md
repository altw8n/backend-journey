# Optional
- контейнер который либо содержит один объект либо не содержит вообще
- stream из 0 или 1 элемента
- способ элегантно работать с ситуациями, когда результат может отсутствовать, вместо того чтобы возвращать null и ловить NullPointerException

```java
//создание
Optional.empty() // пустой
Optional.of(x) // значение точно есть
Optional.ofNullable(x) // значение может быть null
// методы работы с ним
o.get() // проверка наличия значения
```
